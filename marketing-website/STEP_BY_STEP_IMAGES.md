## 🎬 الخطوات بالصور

### الخطوة 1: تحميل الملف ✅ (أنجزت هذه)

```

https://pizoo-landing.preview.emergentagent.com/api/download/marketing-website

📦 اسم الملف: pizoo-website-build.zip
💾 الحجم: 385 KB
```

**ما حدث:**
- ✅ قمت بتحميل الملف بنجاح
- ✅ الملف موجود على جهازك

### الخطوة 2: الذهاب إلى Vercel


### الخطوة 4: رفع الملف

```
سترى صفحة:

┌───────────────────────────────────────┐
│  Import Git Repository                │
│  أو                                   │
│  [Deploy without Git]                │
│                                       │
│  📁 [Browse Files]                   │
└───────────────────────────────────────┘
```

**ماذا تفعل:**
1. تجاهل "Import Git Repository"
2. ابحث عن "Deploy without Git" أو "Browse"
3. انقر عليه
4. اختر الملف `pizoo-website-build.zip` من جهازك

### الخطوة 5: إعدادات المشروع

```
بعد رفع الملف، سترى:

┌───────────────────────────────────────┐
│  Configure Project                    │
│                                       │
│  Project Name:                        │
│  [pizoo-marketing]                   │
│                                       │
│  Framework Preset:                    │
│  [Create React App ▼]                │
│                                       │
│  Root Directory:                      │
│  [./]                                │
│                                       │
│  Build Command: (leave empty)        │
│  Output Directory: [build]           │
│                                       │
│  [Deploy] ← انقر هنا                │
└───────────────────────────────────────┘
```

**ماذا تفعل:**
1. Project Name: اكتب `pizoo-marketing` (أو أي اسم)
2. Framework: اختر "Create React App"
3. Root Directory: اتركه `./`
4. Build Command: اتركه فارغاً
5. Output Directory: تأكد أنه `build`
6. **انقر على [Deploy]**

### الخطوة 10: تحديث DNS

```
بعد إضافة النطاق، Vercel ستعطيك:

┌───────────────────────────────────────┐
│  Configure DNS:                       │
│                                       │
│  Option 1 (Recommended):              │
│  Change nameservers to:               │
│  • ns1.vercel-dns.com                │
│  • ns2.vercel-dns.com                │
│                                       │
│  Option 2 (Manual):                   │
│  Add these DNS records:               │
│  Type: A                              │
│  Name: @                              │
│  Value: 76.76.21.21                  │
└───────────────────────────────────────┘
```

**ماذا تفعل:**
1. اذهب إلى مزود النطاق (GoDaddy, Namecheap, إلخ)
2. ابحث عن "DNS Settings" أو "Nameservers"
3. غيّر Nameservers إلى:
   - `ns1.vercel-dns.com`
   - `ns2.vercel-dns.com`
4. احفظ التغييرات

## 🆘 إذا واجهت مشكلة

### المشكلة: لا أجد "Deploy without Git"
**الحل:** ابحث عن "Import" أو "Browse" أو حاول من:
```
Vercel Dashboard → Overview → New Project → Import
```

### المشكلة: الملف لا يُرفع
**الحل:**
1. تأكد من حجم الملف (385 KB)
2. تأكد من صيغة الملف (.zip)
3. جرّب متصفح آخر

### المشكلة: النطاق لا يعمل بعد 24 ساعة
**الحل:**
1. تحقق من DNS على dnschecker.org
2. في Vercel، اذهب لـ Domains وانقر "Refresh"
3. تأكد من Nameservers صحيحة
